## 规格说明

| 参数 | 设置 |
|------|------|
| 分辨率 | 1920x1080 或 2048x1152 |
| 格式 | .webp |
| 风格后缀 | `digital art, hyper detailed, 8k quality` |
| 构图 | 宽角 (wide angle) 或 全景 (panoramic) |
| 光照 | 戏剧性光照 (dramatic lighting) |

---

## Midjourney 使用方法
在提示词末尾添加风格指令：
```
--ar 16:9 --v 6 --style raw --s 400 --q 2
```

## FLUX 使用方法
直接使用上述提示词即可，无需额外参数

## DALL-E 使用方法
直接使用上述提示词，选择 1024x1024 或 1792x1024
