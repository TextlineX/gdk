## Fish Studio 使用方法

1. 打开 Fish Studio
2. 选择 **Sound Effect** 或 **Voice** 模式
3. 复制下方提示词粘贴到输入框
4. 生成后下载 WAV/MP3
5. 重命名对应文件名，放入 `game/se/`
