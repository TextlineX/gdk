## 快捷生成模板 (Suno)

复制以下格式到 Suno 生成：

```
[Style] Epic cinematic sci-fi ambient orchestral

[Prompt] Shimmering stardust arpeggios, ethereal vocals, cinematic strings, hopeful yet mysterious, cosmic atmosphere, full orchestral swell, emotional piano melody, galaxy background
```

每个 BGM 把 `[Prompt]` 部分替换为对应的中文参考描述即可。
