宇宙的风卷着怪兽墓场的沙砾，刮过贝利亚棱角分明的侧脸。终极战斗仪斜插在地面，暗红色的光映着他眼底化不开的阴霾——自背叛光之国那日起，他的世界便只剩掠夺与复仇，直到GDK的出现，像一颗意外坠入黑暗的星子，撞碎了他冰封多年的心脏。

他第一次见她，是在荒芜的小行星带。她抱着一株发光的水晶草，蹲在陨石上数星星，连他周身翻涌的黑暗气息都未察觉。贝利亚握着战斗仪的手猛地收紧，却没有发动攻击——那双眼睛太干净了，没有对黑暗的恐惧，没有对权力的觊觎，只有对星空的好奇。他鬼使神差地走过去，将一块打磨好的艾美纳尔矿石递到她面前，声音是连自己都陌生的沙哑：“这个，能照亮你走的路。”

从那天起，GDK成了他黑暗人生里唯一的例外。他会收起终极战斗仪的锋芒，陪她坐在陨石上看星云流转；会荡平所有靠近她的异星海盗，然后默默清理掉溅在她裙摆上的血渍；会把怪兽獠牙打磨成守护符，笨拙地系在她手腕上，心里反复演练着“别离开我”，却始终说不出口。他以为这就是爱——用力量为她筑起铜墙铁壁，把她牢牢锁在自己的视线里，这样就不会有人把她抢走，这样她就永远属于自己。可他没看见，GDK望着远方的眼神里，渐渐蒙上了一层疲惫，那是被禁锢的灵魂在无声的挣扎。

他的偏执在一次次失控中发酵。当GDK指着路过的星舰说“那上面的灯光好像光之国”时，他瞬间暴怒，抬手就毁了那艘星舰，看着她震惊的眼神，他又慌了，伸手想去抱她，却被她躲开。那一刻，他心里的黑洞开始扩张，他怕，怕她想起光之国的温暖，怕她看穿自己黑暗里的脆弱，于是只能用更沉重的禁锢来掩饰不安——他把她关在铺满艾美纳尔矿石的宫殿里，以为耀眼的光芒能留住她，却不知，她想要的从来不是囚笼里的奢华。

奶龙的出现，像一把尖刀，刺破了他精心编织的牢笼。那只圆滚滚的黄色小家伙误闯进来，非但没被他的黑暗吓退，还晃着尾巴凑到GDK面前，递出一颗裹着银箔糖纸的星星糖：“你看起来不开心，吃甜的就会笑啦！”GDK接过糖，指尖碰到奶龙温热的爪子，突然笑了——那是贝利亚从未见过的笑容，像解冻的冰河，像破云的阳光，照亮了她眼底的疲惫，也刺痛了贝利亚的心。他第一次意识到，自己给的“守护”，从来不是她想要的温柔。

他开始疯狂地针对奶龙。用黑暗力量掀起风暴，想把这只碍眼的小家伙卷走，可GDK每次都挡在奶龙身前，眼神坚定得像块磐石：“贝利亚，他只是个孩子，你别伤害他！”她的声音里没有往日的顺从，只有抗拒和担忧——担忧的是奶龙，不是他。贝利亚看着她护在奶龙身后的背影，突然觉得自己像个跳梁小丑。他拥有毁天灭地的力量，却留不住一个人的心；他以为给了她全世界最好的一切，却不知，那是她最想挣脱的枷锁。

当GDK牵着奶龙的手转身离去时，贝利亚僵在原地，终极战斗仪掉在地上发出沉闷的声响。他没有追，因为他知道，就算把她抓回来，她的心也不会再属于他。看着他们的身影消失在星云深处，他突然笑了，笑声里全是悲凉，像怪兽墓场里终年不散的风。曾经他以为，黑暗是他的铠甲，却没想到，它成了隔绝他与GDK的墙；他以为占有是爱，却没想到，那是亲手毁掉爱的利刃。

后来，贝利亚又回到了怪兽墓场。终极战斗仪依旧在他手里，依旧能荡平一切阻碍，可他再也没有了复仇的火焰。他常常独自一人站在墓场的顶端，望着GDK离去的方向，周身的黑暗愈发浓重，眼底却藏着一丝连自己都不愿承认的羡慕——羡慕奶龙能给她自由，能陪她看遍星辰大海，而他，只能在无尽的黑暗里，守着那段被自己亲手毁掉的温柔。

他捡起之前给GDK做的守护符，指尖触到上面残留的温度，用力一握，碎片像星尘般散在风中。风里似乎还带着GDK的笑声，和奶龙软糯的声音：“今天我们去看玫瑰星云好不好？”贝利亚闭上眼，黑暗涌入眼底，心里的空洞越来越大。他赢了宇宙，赢了所有对手，却输了唯一的光。

爱情从来不是掠夺与占有，是双向的奔赴与成全。他用黑暗困住了GDK，也困住了自己，最终只能在星尘的错位里，独自承受这份失爱之殇。宇宙的尘埃会掩埋他的悲伤，却抹不掉他心底那道永远愈合不了的伤口——那是他亲手推开的温柔，是他一生都无法释怀的遗憾。
