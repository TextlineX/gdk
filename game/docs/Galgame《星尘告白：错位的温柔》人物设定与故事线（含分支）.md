# Galgame《星尘告白：错位的温柔》人物设定与故事线（含分支）

# 一、核心人物设定（适配galgame互动感，突出可攻略性与情感张力）

## 1. 女主：GDK

**基础信息**：年龄18岁（外表），身份是“星尘守护者”（拥有感知宇宙能量的能力，可微弱操控星尘），无固定居所，漂泊于宇宙各星域，性格前期清冷内敛、温柔敏感，后期逐渐开朗坚定，有自己的原则和底线，厌恶被束缚。

**人物伏笔**：其实是远古星尘族的最后后裔，身上承载着“平衡宇宙明暗力量”的秘密，贝利亚最初接近她，并非单纯一见钟情，而是感知到她身上的星尘力量能压制自身黑暗反噬（后期可解锁相关剧情）；她对“温暖”有着隐秘的渴望，只是被前期的漂泊和贝利亚的禁锢所压抑，潜意识里会被奶龙的软糯气息吸引。

**互动设定**：对奶龙的“呆萌温柔”会触发害羞、指尖微颤等软萌反应（如接过奶龙递来的草莓糖时，耳朵尖会悄悄泛红）；对贝利亚的“偏执守护”会触发抗拒、无奈，后期若选择贝利亚线，会有“深夜陪他看暗域星辰”“抚摸他手背上的黑暗纹路”等试图共情的互动；玩家的选择会影响她的性格走向（是保持清冷疏离，还是变得活泼外放，或是陷入“渴望温暖又害怕束缚”的纠结）。

**关系细节**：与星尘精灵绑定共生，精灵是她童年唯一的陪伴，会在她情绪崩溃时用微光蹭她的脸颊；对贝利亚既有恐惧，也会在看到他独自对着光之国旧徽章发呆时产生隐秘的共情；对奶龙则是从最初的戒备，逐渐依赖他带来的纯粹温暖。

## 2. 男主（可攻略）：奶龙

**基础信息**：年龄外表20岁（实际是宇宙原生“治愈系生命体”，无具体年龄），身份是“星间旅者”，带着一个装满糖果和小零食的帆布背包，四处漂泊，性格呆萌善良、乐观治愈，有点小迷糊，力气不大但韧性极强，擅长用自身能量治愈他人（包括情绪和轻微伤势），对GDK一见钟情，毫无私心。

**人物伏笔**：他的原生能量，恰好是贝利亚黑暗能量的“克星”（但他自己前期不知道），是宇宙孕育出的“平衡因子”；看似呆萌，实则在关键时刻极其可靠，会为了守护GDK，爆发出隐藏的力量——曾在异星荒原用治愈能量逼退三只暴走怪兽，自己却因能量耗尽昏睡三天。

**可攻略亮点**：互动多为“治愈向”，比如给GDK递橘子糖、陪她躺在星尘草地上看流星、在她难过时笨拙地用毛茸茸的脑袋蹭她的手背，选项偏向“温柔陪伴”“尊重选择”，适合喜欢轻松、纯粹爱情的玩家；后期会解锁“治愈贝利亚黑暗能量”的隐藏剧情（若触发相关分支）。

**关系细节**：对贝利亚从最初的恐惧，到在捷德告知贝利亚的过往后，产生“他只是个受伤的人”的共情，会偷偷把糖果放在贝利亚城堡的台阶上；与捷德是忘年交，捷德会暗中告诉他贝利亚的弱点，帮他避开黑暗使者的追捕；背包里永远留着一颗草莓糖，是初见GDK时她无意中说过“星尘的味道像草莓”，便记到现在。

## 3. 男二（可攻略）：贝利亚

**基础信息**：年龄数千岁（奥特曼族，长生），身份是“前黑暗帝王”，曾堕落于黑暗，被光之国放逐后，独自执掌怪兽墓场，性格偏执冷漠、骄傲易怒，掌控欲极强，对GDK是“极致占有+隐秘温柔”，不懂得正确表达爱意，习惯用“力量”守护（实则禁锢）。

**人物伏笔**：他的黑暗能量并非天生，而是被“力量欲望”和“被光之国抛弃的怨恨”滋养；他对GDK的执念，一半是喜欢，一半是因为GDK的星尘力量能“压制他体内的黑暗反噬”（后期可解锁他被光之国审判时的创伤剧情）；内心深处渴望被理解、被接纳，只是不愿低头——曾在GDK睡着后，悄悄把自己珍藏的星尘晶体放在她枕边，误以为能安抚她的情绪。

**可攻略亮点**：互动多为“拉扯向”，比如霸道地把GDK护在身后、吃醋时把奶龙的背包扔出城堡，但会在GDK受伤时不顾一切用黑暗能量击退怪兽，选项偏向“理解他的创伤”“引导他学会尊重”，适合喜欢“偏执疯批美人”“救赎向爱情”的玩家；后期可解锁“贝利亚放弃黑暗，与GDK共生”的结局。

**关系细节**：对捷德既愧疚又别扭，会在捷德被黑暗使者攻击时暗中出手，却从不承认；对黑暗使者的忠诚感到厌恶，认为他们只看重他的黑暗力量，而非他本身；对GDK的占有欲源于“只有她能让我平静”的依赖，后期会学着克制，比如同意她去星尘星域散步；城堡书房的暗格中藏着光之国的旧制服，领口处的磨损是当年被放逐时挣扎留下的痕迹，只有在无人时才会拿出来擦拭。

## 4. 配角（推动剧情/解锁分支）

- 捷德：贝利亚的儿子，立场中立，既理解父亲被抛弃的痛苦，也心疼GDK被禁锢的困境，会在关键节点给玩家提示（如共通线第3章递上光之国旧地图，标记贝利亚被放逐的地点）；解锁他的隐藏剧情需要同时提升奶龙和贝利亚的好感度≥70，剧情中他会说出“我希望父亲能找到救赎，也希望你能自由”的台词；随身带着贝利亚年轻时送的小徽章，表面说“只是个没用的旧物”，实则从不离身。
- 星尘精灵：GDK的守护精灵，能感知GDK的情绪，会在玩家做出“伤害GDK”的选择时（如拒绝奶龙的陪伴），发出尖锐的叫声，同时降低对应角色的好感度；收集齐星尘碎片后，精灵会解锁GDK的星尘族起源剧情；平时喜欢躲在GDK的领口，遇到危险时会化作微光笼罩她，是GDK漂泊岁月里唯一的精神支柱。
- 黑暗使者：贝利亚的旧部，核心成员是“暗黑洛普斯”，不满贝利亚因GDK软化，试图挑拨离间（触发“危机剧情”，如奶龙线中伪装成星尘族人欺骗GDK，贝利亚线中绑架GDK逼贝利亚回归黑暗），他们的目标是夺取GDK的星尘力量，唤醒贝利亚的黑暗面，重新统治暗域；暗黑洛普斯曾是贝利亚最忠诚的部下，因亲眼目睹贝利亚被光之国放逐的惨状，对光之国充满仇恨，坚信只有黑暗才能带来“真正的力量”。

# 二、整体故事线（主线+分支，适配galgame多结局设定）

## 核心世界观

宇宙分为“光域”“暗域”“星尘域”，三者相互制衡：光域代表秩序，由奥特曼族执掌，以光之能量维护宇宙规则；暗域代表力量，由贝利亚及黑暗怪兽掌控，以黑暗能量打破现有平衡；星尘域代表平衡，是远古星尘族的栖息地，星尘能量可中和明暗，维系宇宙稳定。GDK作为星尘族后裔，是维系三者平衡的关键；贝利亚的黑暗能量失控，会导致暗域扩张、光域收缩，宇宙陷入混乱；奶龙的治愈能量，是中和黑暗、修复失衡的核心力量。故事围绕“GDK的身份秘密”“贝利亚的创伤救赎”“奶龙的使命觉醒”展开，玩家的选择决定三人的命运，以及宇宙的最终走向。

## 主线剧情（共8章，前3章为共通线，第4章开始分支）

### 共通线（第1-3章：相遇）
**第1章：星尘漂泊者**
GDK独自漂泊至暗域边缘的荒芜星球，靠吸收星尘能量维持体力，星尘精灵在她肩头发出微弱的光。此时三只暴走怪兽突然袭来，GDK勉强操控星尘抵挡，却因力量不足被怪兽抓伤手臂。危急时刻，奶龙背着帆布背包跌跌撞撞跑来，慌乱中抛出背包里的橘子糖，糖果释放的治愈微光暂时逼退怪兽，他趁机拉着GDK躲进岩缝。
互动选项1：【冷淡道谢后独自离开】（GDK性格偏向清冷，奶龙好感-5）
互动选项2：【接过他递来的草莓糖，轻声说谢谢】（GDK触发害羞反应，奶龙好感+10，星尘精灵发出愉悦的嗡鸣）
两人短暂交流后，奶龙得知GDK在寻找星尘碎片，主动提出同行，称自己见过类似的碎片。此时贝利亚的黑暗能量突然笼罩星球，他瞬移至岩缝前，盯着GDK手臂上的星尘印记，语气冰冷地说：“跟我走。”
互动选项1：【抗拒后退，躲到奶龙身后】（贝利亚好感-5，奶龙好感+5）
互动选项2：【沉默着站在原地，没有反抗】（贝利亚好感+5，奶龙好感-5）
无论选择哪个选项，贝利亚都会用黑暗能量将GDK带回怪兽墓场的城堡，留下一脸焦急的奶龙。

**第2章：城堡囚笼**
GDK被安置在城堡顶层的房间，房间里摆满了贝利亚收集的星尘晶体，却没有一扇窗户。星尘精灵因黑暗能量的压迫变得萎靡，GDK尝试操控星尘打开房门，却被贝利亚设下的黑暗屏障反弹。贝利亚走进房间，递给她一瓶治疗伤口的药剂，语气生硬地说：“这里很安全，别想着离开。”
互动选项1：【打翻药剂，质问他“你把我关在这里到底想干什么”】（贝利亚好感-10，触发他的愤怒剧情，房门被加锁）
互动选项2：【接过药剂，轻声问“你认识星尘族吗”】（贝利亚好感+5，他会停顿片刻，回答“很久以前见过”）
夜晚，GDK睡不着，走到城堡露台，看到贝利亚独自对着光之国旧徽章发呆。她犹豫着靠近，贝利亚察觉到动静，立刻收起徽章，变回冷漠的样子。此时捷德突然出现，递给GDK一张光之国旧地图，低声说：“这上面标记着贝利亚被放逐的地点，或许能帮你理解他。”
互动选项1：【接过地图，向捷德道谢】（解锁捷德隐藏剧情前置条件）
互动选项2：【拒绝地图，说“我不需要理解他”】（捷德好感-5）

**第3章：闯入的阳光**
奶龙在捷德的帮助下，避开黑暗使者的追捕，偷偷潜入城堡。他找到GDK的房间，用治愈能量弱化黑暗屏障，带着GDK逃到城堡外的星尘草地。两人躺在草地上看流星，奶龙拿出背包里的糖果，笑着说：“我听说流星能实现愿望，我希望你能永远开心。”
互动选项1：【靠在他的肩膀上，说“谢谢你”】（奶龙好感+15，GDK性格开始偏向开朗）
互动选项2：【沉默地看着流星，说“我没有愿望”】（奶龙好感+5，GDK性格保持清冷）
此时贝利亚追来，看到两人亲密的样子，愤怒地将奶龙的背包扔出很远，黑暗能量在他周身涌动。GDK立刻挡在奶龙身前，对着贝利亚大喊：“不要伤害他！”
互动选项1：【拉住贝利亚的手，说“我跟你回去，但你不许伤害他”】（进入贝利亚线，贝利亚好感+10，奶龙好感-5）
互动选项2：【拉着奶龙的手跑开，说“我们走”】（进入奶龙线，奶龙好感+10，贝利亚好感-5）
若两人好感度均≥60，且接过捷德的地图，会触发隐藏选项【站在两人中间，说“我们谈谈”】，进入平衡线（解锁捷德隐藏剧情）。

### 分支剧情（第4-8章）
#### 奶龙线：星尘与暖阳
**核心逻辑**：以“治愈与守护”为核心，奶龙逐渐觉醒平衡因子的使命，GDK在纯粹的温暖中打开心扉，两人共同治愈贝利亚的黑暗创伤，维护宇宙平衡。
**第4章：星尘碎片的秘密**
奶龙带着GDK来到星尘域边缘，找到第一块星尘碎片。碎片释放出星尘族的记忆影像，显示星尘族因试图平衡明暗力量被光之国和暗域联手剿灭。GDK情绪崩溃，奶龙用治愈能量安抚她，星尘精灵也发出微光蹭她的脸颊。此时黑暗使者伪装成星尘族人，欺骗GDK说“贝利亚是剿灭星尘族的凶手”。
互动选项1：【相信黑暗使者，对贝利亚产生恨意】（触发危机剧情，贝利亚后期会被黑暗使者控制）
互动选项2：【质疑黑暗使者，说“我需要证据”】（解锁奶龙隐藏剧情，奶龙的治愈能量能识破黑暗伪装）
选择选项2后，奶龙的背包里突然发出微光，治愈能量驱散黑暗使者的伪装，两人击退黑暗使者，得知贝利亚当年曾试图阻止星尘族被剿灭，却被光之国视为叛徒。

**第5章：使命觉醒**
捷德找到两人，告知贝利亚的黑暗能量即将失控，宇宙即将失衡。奶龙在星尘碎片的引导下，觉醒平衡因子的使命，他的治愈能量能直接中和黑暗能量。GDK决定和奶龙一起前往怪兽墓场，帮助贝利亚控制黑暗能量。
互动选项1：【主动握住奶龙的手，说“我们一起去”】（GDK性格完全开朗，奶龙好感+10）
互动选项2：【犹豫着说“我怕贝利亚不接受”】（GDK性格保持温柔敏感，奶龙好感+5）

**第6章：城堡对峙**
两人来到怪兽墓场，贝利亚因黑暗能量反噬陷入痛苦，黑暗使者趁机发动攻击，试图夺取GDK的星尘力量。奶龙爆发出治愈能量，击退黑暗使者，却因能量耗尽晕倒。GDK用星尘力量压制贝利亚的黑暗反噬，贝利亚清醒后，看到晕倒的奶龙，沉默着帮他治疗伤口。
互动选项1：【对贝利亚说“谢谢你”】（贝利亚好感+5，解锁后期治愈剧情）
互动选项2：【抱着奶龙，说“我们走”】（贝利亚好感-5，触发贝利亚独自对抗黑暗使者剧情）

**第7章：三方和解**
捷德召集光之国的部分奥特曼，解释星尘族的真相，光之国同意暂时放下对贝利亚的仇恨。GDK、奶龙、贝利亚联手，前往暗域核心，清除黑暗使者的势力。奶龙用治愈能量修复暗域的失衡，GDK用星尘力量稳定宇宙，贝利亚则用黑暗力量镇压残余的暴走怪兽。

**第8章：星尘告白**
宇宙恢复平衡后，GDK和奶龙回到星尘草地，奶龙拿出一颗用星尘碎片做成的草莓糖，笑着说：“我喜欢你，不是因为使命，是因为你就是你。”GDK接过糖果，耳朵尖泛红，轻声说：“我也是。”
**结局1：暖阳相伴**（好感度≥90）：两人成为星间旅者，带着星尘精灵四处漂泊，偶尔会去怪兽墓场看望贝利亚和捷德，画面定格在三人坐在城堡露台看流星的瞬间，情绪落点是“纯粹温暖的圆满”。
**结局2：平衡守护者**（好感度70-89）：GDK成为星尘域的守护者，奶龙留在星尘域陪伴她，两人共同维护宇宙平衡，画面定格在星尘族遗址前，GDK操控星尘，奶龙在一旁微笑，情绪落点是“责任与爱情的平衡”。
**结局3：遗憾的陪伴**（好感度<70）：奶龙完成使命后回到宇宙深处，GDK继续漂泊，偶尔会收到奶龙寄来的糖果，画面定格在GDK看着星空，手里握着草莓糖，情绪落点是“释然的牵挂”。

#### 贝利亚线：黑暗与星尘
**核心逻辑**：以“救赎与拉扯”为核心，GDK逐渐理解贝利亚的创伤，引导他放下仇恨，两人用星尘力量和黑暗力量的结合，维护宇宙平衡，实现相互救赎。
**第4章：黑暗的创伤**
